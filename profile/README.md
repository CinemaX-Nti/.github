## About CinemaX

CinemaX is a modern cinema booking and management system designed to deliver a premium digital experience for both customers and admins.

Users can:

- Discover featured and now-showing movies
- View movie details and trailers
- Choose showtimes and reserve seats
- Add snacks and drinks before checkout
- Complete bookings through a guided flow

Admins can:

- Manage movies, halls, and showtimes
- Track bookings and payment reviews
- Manage snacks and drinks
- Monitor platform activity from the dashboard

## Project Highlights

| Feature | Description |
|---|---|
| Customer Journey | Landing page, movies, showtimes, seats, snacks, checkout |
| Authentication | Email/password login, Google sign-in, OTP-based flows |
| Admin Dashboard | Manage movies, halls, menu items, bookings, and users |
| Seat Experience | Visual seat map with standard, premium, and VIP pricing |
| Architecture | Angular frontend with Express and MongoDB backend |
